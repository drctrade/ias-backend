\# IAS Backend API



Serveur Node.js pour le scraping et la génération de packages automatiques.



\## Endpoints



\- `GET /` - Health check

\- `POST /api/scrape/website` - Scraper un site web

\- `POST /api/generate/package` - Générer un package complet

\- `GET /api/prospects` - Récupérer les prospects



\## Déploiement



Déployé sur Render.com
